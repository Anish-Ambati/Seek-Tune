Backend: FastAPI + Python
Upload full songs to save fingerprints, and short clips to identify tracks.

0. Download from Spotify

Paste a Spotify track link. The system will download, convert, and fingerprint it automatically.



1. Save Song

Select a full song file (WAV/MP3/etc.) and upload it to generate fingerprints.


2. Find Song

Select a short audio clip. The system will try to recognize the song.


API Info

Backend base URL: http://localhost:8000
Endpoints: POST /api/save, POST /api/find

Make sure the backend is running: python main.py serve --port 8000
